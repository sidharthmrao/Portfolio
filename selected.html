AutoBoard Choate Robotics

Previous Next

Robot auto-pathing simulator written in Python and PyGame which our FRC team used to create and simulate better autonomous routines on the fly. Allowed us to input different routines, visualize how they run and whether they hit obstacles, move waypoints around to generate new paths, and showed information about robot velocity, trajectory, etc. so we could understand if movements weren't realistically feasible for the robot.
